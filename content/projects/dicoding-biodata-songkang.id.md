# 🎭 Song Kang — Website Biodata

Website biografi sederhana tentang aktor Korea **Song Kang**, dibangun dengan HTML dan CSS murni sebagai bagian dari kursus Dicoding *Belajar Dasar Pemrograman Web*. Ini adalah salah satu proyek web pertama saya — fokus pada mempelajari dasar-dasar struktur HTML dan styling CSS.

## 💡 Konsep Proyek

Tujuan proyek ini adalah berlatih membangun halaman web multi-bagian menggunakan HTML semantik dan CSS dasar. Song Kang dipilih sebagai subjek karena popularitasnya sebagai aktor Korea — menjadikannya topik yang menyenangkan untuk dikerjakan sambil mempelajari dasar-dasarnya.

Situs ini disusun seperti halaman biografi penggemar sederhana, mencakup tempat kelahirannya, garis waktu karir, dan pencapaian.

## ✨ Fitur Utama

- **Bagian Tempat Kelahiran** — Informasi dan gambar Suwon, Korea Selatan — tempat lahir Song Kang (lahir 23 April 1994).
- **Bagian Karir** — Garis waktu karir akting Song Kang, dimulai dari debutnya di *The Liar and His Lover* (2017) dengan agensi Namoo Actors.
- **Bagian Pencapaian** — Sorotan peran drama terkenal dan pencapaian karirnya.
- **Navbar** — Navbar berbasis anchor yang menghubungkan ke setiap bagian — Tempat Kelahiran, Karir Songkang, dan Pencapaian Songkang.
- **Google Fonts** — Menggunakan keluarga font *Quicksand* untuk tampilan yang bersih dan modern.

## 🛠️ Teknologi yang Digunakan

| Teknologi | Deskripsi |
|---|---|
| HTML5 | Struktur halaman dan elemen semantik |
| CSS3 | Styling, layout, warna, dan tipografi |
| Google Fonts | Font Quicksand via CDN |

## 🎯 Tujuan

Proyek ini dibangun sebagai **submission Dicoding** untuk kursus *Belajar Dasar Pemrograman Web*. Ini adalah salah satu percobaan pertama saya membangun halaman web nyata dari nol — berlatih elemen HTML seperti `header`, `nav`, `main`, `article`, `section`, dan CSS dasar untuk layout dan styling.
