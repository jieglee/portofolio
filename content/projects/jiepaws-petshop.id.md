# 🐾 JiePaws — Toko Hewan Peliharaan

Aplikasi web e-commerce toko hewan peliharaan yang berfungsi lengkap, dibangun dengan React + Vite dan Tailwind CSS. Dibuat sebagai proyek sekolah (Kelas X, Semester 2) untuk konsentrasi Rekayasa Perangkat Lunak di SMK Taruna Bhakti.

## 💡 Konsep Proyek

JiePaws adalah platform toko hewan peliharaan online di mana pengguna dapat menjelajahi produk hewan, menambahkan item ke keranjang atau wishlist, menyelesaikan alur checkout, dan mengelola pesanan serta profil — semua dalam aplikasi web modern dan responsif dengan animasi Framer Motion yang mulus.

## ✨ Fitur Utama

- **Landing Page** — Halaman publik dengan bagian hero, produk bestseller, tampilan brand, dan footer.
- **Autentikasi** — Alur login dan register lengkap dengan proteksi route. Pengguna dialihkan ke home setelah login berhasil.
- **Halaman Home** — Homepage yang dipersonalisasi dengan search bar, bagian promo, kategori hewan, grid bestseller, dan carousel brand.
- **Detail Produk** — Halaman produk individu dengan detail lengkap dan tombol Tambah ke Keranjang.
- **Keranjang** — Halaman keranjang yang menampilkan item terpilih dengan kontrol kuantitas dan total harga.
- **Checkout** — Alur checkout untuk menyelesaikan pembelian dari keranjang.
- **Wishlist** — Simpan produk favorit ke wishlist pribadi.
- **Pesanan Saya** — Halaman riwayat pesanan yang menampilkan pembelian sebelumnya.
- **Profil** — Halaman profil pengguna dengan pengelolaan informasi pribadi.
- **Notifikasi Toast** — Feedback real-time menggunakan `react-toastify` untuk aksi keranjang, login berhasil, dan lainnya.

## 🛠️ Teknologi yang Digunakan

| Teknologi | Deskripsi |
|---|---|
| React | Library UI berbasis komponen |
| Vite | Build tool dan dev server cepat |
| Tailwind CSS | CSS framework utility-first |
| Framer Motion | Animasi halaman dan komponen yang mulus |
| React Router DOM | Routing sisi klien |
| React Toastify | Sistem notifikasi toast |

## 🚀 Memulai

```bash
git clone https://github.com/jieglee/jiepaws-react-vite.git
cd jiepaws-react-vite
npm install
npm run dev
```

## 📜 Script yang Tersedia

```bash
npm run dev
npm run build
npm run preview
npm run lint
npm run start
```
