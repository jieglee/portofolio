# 🎬 MyTix — Cinema Ticket Landing Page

> "No Antri No Ribet No Drama — #SayGoodbyeToAntri"

MyTix is a cinema ticket booking landing page concept designed to showcase a modern, dark-themed UI for an online movie ticketing platform. Built as a frontend-only project using HTML and Tailwind CSS, it demonstrates responsive layout, component design, and smooth UI interactions without any backend.

## 💡 Project Concept

MyTix was created as a UI concept for a cinema ticketing app — inspired by real platforms like TIX ID and CGV. The goal was to design a visually appealing, mobile-friendly landing page that communicates the core value proposition: book tickets online, skip the queue.

The design uses a dark black/deep purple color scheme with yellow accents to evoke the cinematic experience.

## ✨ Key Features

**🎥 Hero Section**
Full-screen background with blur effect and bold tagline — "No Antri No Ribet No Drama" with an animated hashtag highlight.

**🍿 Now Showing**
A responsive movie grid showcasing currently playing films with poster thumbnails and titles.

**📢 Ads Banner Carousel**
Auto-playing image slider built with Swiper.js, featuring promotional banners.

**🏢 Partner Cinemas**
Logo showcase of partner cinema chains: XXI, CGV, Cinepolis.

**💳 Payment Methods**
Supported payment options displayed: GoPay, OVO, DANA.

**📱 Responsive Navbar**
Desktop navbar with centered menu + hamburger menu for mobile with smooth toggle.

**🎞️ Movie Categories**
Sections for browsing movies by category — now showing, coming soon, and featured titles.

## 🛠️ Technologies Used

| Technology | Description |
|---|---|
| 🧱 HTML5 | Page structure and all sections |
| 🎨 Tailwind CSS | Utility-first styling via CDN |
| ⚙️ JavaScript (Vanilla) | Mobile menu toggle and interactivity |
| 🔄 Swiper.js | Touch-friendly image carousel for ads banner |
