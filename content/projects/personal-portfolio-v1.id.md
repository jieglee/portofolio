# 💜 Personal Portfolio v1

Situs portofolio pribadi yang dibangun dari nol menggunakan HTML, CSS, dan JavaScript murni. Ini adalah proyek web pertama saya — dirancang untuk memperkenalkan diri sebagai Frontend Developer dan UI/UX Designer, lengkap dengan showcase skill, galeri proyek, dan formulir kontak. Dibuat sebagai proyek sekolah (Kelas X, Semester 1) untuk konsentrasi Rekayasa Perangkat Lunak di SMK Taruna Bhakti.

## ✨ Fitur

- **Typed.js Intro** — Efek ketik animasi pada bagian hero yang berganti antara "Frontend Developer" dan "UI/UX Designer".
- **Hero Section** — Perkenalan diri dengan foto profil dan tombol unduh CV.
- **Skills Section** — Grid visual yang menampilkan 6 tools — HTML, CSS, JavaScript, Adobe Illustrator, Figma, dan Python.
- **Projects Section** — Galeri 4 karya awal dari Kelas 10, Semester 1:
  - **Proyek 1** — Website sederhana dengan HTML & CSS (proyek sertifikat Dicoding)
  - **Proyek 2** — Ilustrasi Hatsune Miku menggunakan Adobe Illustrator
  - **Proyek 3** — Rekreasi logo Gmail menggunakan Adobe Illustrator
  - **Proyek 4** — Desain UI/UX untuk aplikasi pemesanan tiket bioskop di Figma
- **Contact Section** — Formulir kontak menggunakan Formsubmit, dengan kolom nama, email, dan pesan.
- **Footer** — Tautan media sosial ke Instagram dan GitHub menggunakan ikon Font Awesome.
- **Desain Responsif** — Beradaptasi untuk ukuran layar desktop dan mobile.
- **Navigasi Halus** — Navbar berbasis anchor untuk scrolling mulus antar bagian.

## 🚀 Teknologi yang Digunakan

| Teknologi | Deskripsi |
|---|---|
| HTML5 | Menyusun semua bagian dan konten halaman |
| CSS3 | Layout, styling, animasi, dan desain responsif |
| JavaScript (Vanilla JS) | Manipulasi DOM dan perilaku dinamis |
| Typed.js | Efek ketik animasi di bagian hero (via CDN) |
| Font Awesome | Ikon di navbar, footer, dan tautan media sosial (via CDN) |
| Formsubmit | Menangani pengiriman formulir kontak tanpa backend |

## 🔧 Cara Kerja

1. Buka `index.html` di browser modern mana pun — tidak perlu build tools
2. Gunakan navbar untuk scroll ke bagian Home, Skill, Project, atau Contact Me
3. Animasi Typed.js secara otomatis berganti melalui judul role di bagian hero
4. Klik **Download CV** di bagian hero untuk mengunduh CV sebagai PDF
5. Jelajahi bagian Projects untuk melihat setiap karya dengan deskripsi dan tautan langsung
6. Isi formulir kontak — pengiriman dikirim via Formsubmit langsung ke email
7. Tautan media sosial di navbar dan footer mengarah ke profil Instagram dan GitHub

## 📦 Instalasi & Penggunaan

```bash
# Tidak perlu instalasi!
# Cukup clone dan buka index.html di browser kamu
git clone https://github.com/jieglee/deploy-portofolio.git
```

Kemudian buka `index.html` secara langsung — tanpa build tools, tanpa npm install.
