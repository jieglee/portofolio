# 🐾 JiePaws — Pet Shop E-Commerce

A fully functional pet shop e-commerce web app built with React + Vite and Tailwind CSS. Created as a school project (Grade X, Semester 2) for the Software Engineering concentration at SMK Taruna Bhakti.

## 💡 Project Concept

JiePaws is an online pet shop platform where users can browse pet products, add items to cart or wishlist, complete a checkout flow, and manage orders and profile — all in a modern, responsive web app with smooth Framer Motion animations.

## ✨ Key Features

- **🏠 Landing Page** — Public-facing page with hero section, bestseller products, brand showcase, and footer.
- **🔐 Authentication** — Full login and register flow with route protection. Users are redirected to home after successful login.
- **🐶 Home Page** — Personalized homepage with search bar, promo section, pet categories, bestseller grid, and brand carousel.
- **🛍️ Product Detail** — Individual product page with full details and Add to Cart button.
- **🛒 Cart** — Cart page showing selected items with quantity control and total price.
- **💳 Checkout** — Checkout flow for completing purchases from the cart.
- **❤️ Wishlist** — Save favorite products to a personal wishlist.
- **📦 My Orders** — Order history page showing past purchases.
- **👤 Profile** — User profile page with personal information management.
- **🔔 Toast Notifications** — Real-time feedback using `react-toastify` for cart actions, login success, and more.

## 🛠️ Technologies Used

| Technology | Description |
|---|---|
| ⚛️ React | Component-based UI library |
| ⚡ Vite | Fast build tool and dev server |
| 🎨 Tailwind CSS | Utility-first CSS framework |
| 🎞️ Framer Motion | Smooth page and component animations |
| 🔀 React Router DOM | Client-side routing |
| 🔔 React Toastify | Toast notification system |

## 🚀 Getting Started

```bash
git clone https://github.com/jieglee/jiepaws-react-vite.git
cd jiepaws-react-vite
npm install
npm run dev
```

## 📜 Available Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
npm run start
```
