# 🎬 MyTix — Landing Page Tiket Bioskop

> "No Antri No Ribet No Drama — #SayGoodbyeToAntri"

MyTix adalah konsep landing page pemesanan tiket bioskop yang dirancang untuk menampilkan UI bertema gelap modern untuk platform pemesanan tiket film online. Dibangun sebagai proyek frontend-only menggunakan HTML dan Tailwind CSS, proyek ini mendemonstrasikan layout responsif, desain komponen, dan interaksi UI yang mulus tanpa backend.

## 💡 Konsep Proyek

MyTix dibuat sebagai konsep UI untuk aplikasi pemesanan tiket bioskop — terinspirasi oleh platform nyata seperti TIX ID dan CGV. Tujuannya adalah merancang halaman landing yang menarik secara visual, ramah mobile, yang mengkomunikasikan proposisi nilai utama: pesan tiket online, lewati antrean.

Desain menggunakan skema warna ungu gelap/hitam dengan aksen kuning untuk menciptakan pengalaman sinematik.

## ✨ Fitur Utama

- **Hero Section** — Latar belakang layar penuh dengan efek blur dan tagline tebal — "No Antri No Ribet No Drama" dengan sorotan hashtag animasi.
- **Now Showing** — Grid film responsif yang menampilkan film yang sedang tampil dengan poster dan judul.
- **Ads Banner Carousel** — Slider gambar auto-play menggunakan Swiper.js, menampilkan banner promosi.
- **Partner Cinemas** — Tampilan logo jaringan mitra bioskop: XXI, CGV, Cinepolis.
- **Payment Methods** — Opsi pembayaran yang didukung: GoPay, OVO, DANA.
- **Navbar Responsif** — Navbar desktop dengan menu tengah + hamburger menu untuk mobile dengan toggle mulus.
- **Kategori Film** — Bagian untuk menjelajahi film berdasarkan kategori — sedang tampil, segera hadir, dan film unggulan.

## 🛠️ Teknologi yang Digunakan

| Teknologi | Deskripsi |
|---|---|
| HTML5 | Struktur halaman dan semua bagian |
| Tailwind CSS | Utility-first styling via CDN |
| JavaScript (Vanilla) | Toggle menu mobile dan interaktivitas |
| Swiper.js | Carousel gambar sentuh untuk banner iklan |
